## 소개

인터뷰 중계 프로세스를 제공하여

전문가와 리포터 사이의 컨택 시간을 유의미하게 단축해 주는 서비스

## 주요 기능

- 전문가와 리포터 회원 구분
- 인터뷰 요청 프로세스
- 실시간 인터뷰 기한 마감
